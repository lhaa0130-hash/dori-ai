# 개요 및 개발 배경

## 프로젝트 소개

APPLICATION: DORI는 모바일 환경에서 더 나은 사용자 경험을 제공하기 위해 개발된 Android 애플리케이션입니다.

## 개발 배경

- 모바일 사용자 증가에 따른 네이티브 앱 필요성
- 웹 기반 서비스의 한계 극복
- 오프라인 기능 제공
- 푸시 알림 및 실시간 업데이트 지원

## 주요 목표

- 직관적인 사용자 인터페이스
- 빠른 성능과 안정성
- 다양한 Android 기기 호환성
