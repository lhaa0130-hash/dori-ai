AI 그림 프롬프트 작성법: 원하는 이미지를 뽑는 단계별 입력 공식

2026년 7월 1일AM 08:02 by illo
공유하기

AI 그림 프롬프트 작성법을 검색했다면 대부분 “무엇을 어떻게 써야 원하는 그림이 나오는지”가 가장 막막할 것입니다. 같은 도구를 써도 어떤 사람은 포스터처럼 선명한 이미지를 만들고, 어떤 사람은 어색한 손·흐린 배경·엉뚱한 구도가 나옵니다. 차이는 재능보다 프롬프트를 구조화하는 방식에 있습니다.

아래 가이드는 초보자가 바로 따라 할 수 있도록 도구 선택 설정 입력 문장 작성 수정 저장 순서로 정리했습니다. Midjourney, DALL·E 계열, Gemini, Microsoft Designer, Stable Diffusion 계열 도구에 공통으로 적용할 수 있습니다.

1단계: AI 그림 도구를 먼저 고르기

AI 그림 프롬프트 작성법은 도구마다 조금씩 달라집니다. 어떤 도구는 자연어 설명에 강하고, 어떤 도구는 세부 옵션과 부정 프롬프트를 잘 활용해야 합니다.

도구 무료 사용 여부 장점 아쉬운 점 추천 대상
Microsoft Designer / Copilot 이미지 생성 Microsoft Copilot은 무료로 사용할 수 있으나, 일부 기능은 로그인해야 접근할 수 있고 사용량 제한이 있을 수 있음 접근이 쉽고 한국어 입력도 비교적 편함 세밀한 스타일 고정은 한계가 있을 수 있음 입문자, 블로그 썸네일 제작
Gemini 이미지 생성 계정·지역·연령·요금제와 기능 종류에 따라 제공 범위가 다를 수 있음 대화형으로 수정 요청하기 쉬움 일부 생성 기능은 별도 요금제나 적합한 라이선스가 필요할 수 있음 설명하면서 이미지를 다듬고 싶은 사람
ChatGPT 이미지 생성 계열 제공 범위와 사용 조건은 서비스 정책에 따라 확인 필요 문장형 요청과 수정 대화에 강함 생성 제한이 있을 수 있음 기획안, 광고 이미지, 콘셉트 시안
Midjourney 이용 조건과 요금제는 공식 안내 확인 필요 예술적 완성도와 스타일 표현이 강함 초보자는 명령어와 옵션이 낯설 수 있음 포스터, 캐릭터, 고퀄리티 콘셉트 아트
Stable Diffusion 계열 로컬 설치, 웹 서비스, 배포 모델에 따라 조건 확인 필요 모델·로라·부정 프롬프트 등 세밀한 제어 설치형은 난도가 높음 세밀한 커스터마이징이 필요한 사용자

처음이라면 Microsoft Designer, Gemini, ChatGPT 이미지 생성 계열처럼 대화형으로 수정하기 쉬운 도구를 추천합니다. “한 번에 완벽한 프롬프트”보다 “생성 후 수정”이 훨씬 빠릅니다.

2단계: 생성 전에 용도와 비율을 정하기

프롬프트를 쓰기 전에 이미지를 어디에 쓸지부터 정해야 합니다. 용도가 없으면 AI도 구도와 여백을 제대로 잡기 어렵습니다.

예를 들어 같은 “커피 컵 사진”이라도 용도에 따라 프롬프트가 달라집니다.

  • 블로그 썸네일: 상단에 제목을 넣을 여백, 밝은 배경
  • 인스타그램 게시물: 정사각형, 감성적인 색감
  • 유튜브 썸네일: 강한 대비, 큰 피사체, 시선이 가는 구도
  • 쇼핑몰 상세페이지: 제품 중심, 흰 배경, 왜곡 최소화
  • 포스터: 세로 비율, 타이포그래피 공간, 극적인 조명

도구에 비율 설정이 있다면 먼저 선택하세요. 보통은 1:1 정사각형, 16:9 가로형, 9:16 세로형, 4:5 SNS형 중 하나를 고르면 됩니다. 비율 옵션이 없다면 프롬프트에 직접 넣습니다.

예시:

세로형 포스터 비율, 9:16, 중앙에 인물이 있고 상단에는 제목을 넣을 수 있는 빈 공간

3단계: AI 그림 프롬프트 작성법의 기본 공식 익히기

가장 안정적인 프롬프트 구조는 다음 순서입니다.

주제 + 행동/상황 + 배경 + 스타일 + 구도 + 조명 + 색감 + 품질 조건 + 제외할 요소

초보자는 이 공식을 그대로 채우면 됩니다.

기본 템플릿

[주제]가 [행동/상황]을 하고 있는 장면, [배경], [스타일], [구도], [조명], [색감], [품질 조건], [제외할 요소]

예시:

고양이 모양의 작은 로봇이 책상 위에서 노트북을 보고 있는 장면, 따뜻한 홈오피스 배경, 귀여운 3D 일러스트 스타일, 정면에서 약간 내려다본 구도, 부드러운 아침 햇살, 파스텔톤 색감, 선명하고 깔끔한 이미지, 글자와 로고는 제외

나쁜 프롬프트는 보통 너무 짧습니다.

  • 나쁜 예: “예쁜 카페 그림”
  • 좋은 예: “비 오는 저녁의 작은 골목 카페, 창가에 따뜻한 조명이 켜져 있고 테이블 위에 라떼 한 잔이 놓인 장면, 감성적인 디지털 일러스트, 세로형 포스터 구도, 부드러운 빛 번짐, 차분한 브라운과 오렌지 색감, 사람 얼굴은 나오지 않게”

핵심은 AI가 추측하지 않게 하는 것입니다. “예쁘게”보다 “따뜻한 조명, 파스텔톤, 미니멀한 배경”처럼 눈에 보이는 단어로 바꾸세요.

4단계: 주제는 구체적으로, 형용사는 시각적으로 쓰기

AI 그림은 추상적인 감정보다 시각적 단서에 강합니다. “고급스러운”이라고만 쓰면 결과가 흔들립니다. 대신 고급스러움을 만드는 요소를 적어야 합니다.

추상 표현 더 좋은 시각 표현
고급스러운 대리석 배경, 금색 포인트, 절제된 조명, 미니멀한 구도
귀여운 둥근 형태, 큰 눈, 파스텔 색감, 작은 몸 비율
미래적인 유리 건물, 네온 라이트, 홀로그램 UI, 금속 질감
따뜻한 노란 조명, 나무 테이블, 석양빛, 부드러운 그림자
전문적인 흰 배경, 정돈된 레이아웃, 선명한 제품 사진, 과한 장식 없음

예시 프롬프트:

프리미엄 스킨케어 제품 광고 이미지, 흰색 세럼 병 1개가 대리석 받침대 위에 놓여 있음, 배경은 밝은 베이지톤, 부드러운 스튜디오 조명, 미니멀한 구도, 제품 라벨은 비워두기, 깨끗하고 고급스러운 상업 사진 스타일

이렇게 쓰면 “고급스럽게”보다 훨씬 안정적인 결과가 나옵니다.

5단계: 구도·카메라·조명을 넣어 완성도 높이기

프롬프트 초보자가 가장 많이 빼먹는 부분이 구도와 조명입니다. 하지만 실제 이미지의 완성도는 여기서 크게 갈립니다.

자주 쓰는 구도 표현은 다음과 같습니다.

  • 정면 구도: 제품, 프로필, 캐릭터 시트에 적합
  • 상단에서 내려다본 구도: 음식, 책상, 문구류, 플랫레이 이미지에 적합
  • 클로즈업: 감정, 질감, 제품 디테일 강조
  • 와이드샷: 풍경, 공간, 배경이 중요한 장면에 적합
  • 중앙 배치: 썸네일, 광고, 상세페이지에 안정적
  • 좌측 여백 / 우측 여백: 텍스트를 얹을 디자인에 유용

조명 표현도 함께 넣습니다.

  • 부드러운 자연광
  • 스튜디오 조명
  • 역광
  • 네온 조명
  • 따뜻한 석양빛
  • 강한 명암 대비
  • 그림자가 거의 없는 밝은 조명

예시:

건강한 샐러드 볼 사진, 상단에서 내려다본 플랫레이 구도, 흰색 테이블 위에 신선한 채소와 닭가슴살, 자연광, 밝고 깨끗한 색감, 블로그 썸네일용, 오른쪽에 텍스트를 넣을 빈 공간

6단계: 스타일은 1~2개만 명확히 지정하기

스타일을 너무 많이 넣으면 결과가 섞여 어색해질 수 있습니다. “수채화, 픽사풍, 실사, 애니메이션, 사이버펑크, 미니멀”처럼 한 번에 여러 스타일을 넣기보다 목적에 맞게 1~2개만 고르세요.

자주 쓰는 스타일 예시는 다음과 같습니다.

  • 실사 사진 스타일
  • 3D 일러스트 스타일
  • 수채화 일러스트
  • 미니멀 벡터 일러스트
  • 애니메이션 배경화 스타일
  • 레트로 포스터 스타일
  • 제품 광고 사진 스타일
  • 패션 화보 스타일
  • 아이소메트릭 일러스트

예시:

온라인 강의 플랫폼 홍보용 이미지, 노트북 화면을 보며 공부하는 20대 인물, 밝은 방 안, 미니멀 3D 일러스트 스타일, 파란색과 흰색 중심의 색감, 정사각형 구도, 친근하고 깔끔한 분위기

주의할 점은 특정 생존 작가나 저작권이 있는 캐릭터의 스타일을 그대로 요구하는 방식은 피하는 것이 좋습니다. 대신 “따뜻한 수채화풍”, “고전 만화 느낌”, “미래적인 네온 도시”처럼 일반적인 미술·디자인 용어로 바꾸는 편이 안전합니다.

7단계: 부정 프롬프트로 실패 요소 줄이기

일부 도구는 Negative Prompt 입력칸을 제공합니다. 없다면 본문 프롬프트 끝에 “다음 요소는 제외”라고 적어도 됩니다.

자주 제외하는 요소는 다음과 같습니다.

  • 글자 깨짐
  • 로고
  • 워터마크
  • 흐릿함
  • 왜곡된 손
  • 여분의 손가락
  • 비대칭 얼굴
  • 과도한 노이즈
  • 저해상도 느낌
  • 잘린 피사체
  • 복잡한 배경

예시:

제외할 요소: 흐릿한 이미지, 깨진 글자, 로고, 워터마크, 손가락 왜곡, 과하게 복잡한 배경, 잘린 얼굴

단, 부정 프롬프트가 만능은 아닙니다. 특히 손, 글자, 복잡한 포즈는 여전히 오류가 날 수 있으므로 처음부터 “손이 화면 밖에 있음”, “텍스트 없음”, “단순한 포즈”처럼 설계하는 것이 더 안정적입니다.

8단계: 한 번에 끝내지 말고 수정 요청하기

AI 그림 프롬프트 작성법에서 가장 중요한 습관은 첫 결과를 평가하고 다시 지시하는 것입니다. 처음 프롬프트로 70%를 만들고, 수정 프롬프트로 90%까지 끌어올린다고 생각하면 됩니다.

수정 요청은 구체적으로 해야 합니다.

  • “더 예쁘게” 모호함
  • “배경을 더 밝은 베이지톤으로 바꾸고, 제품은 중앙에 더 크게 배치해줘” 좋음
  • “인물 표정을 자연스럽게 하고, 손은 보이지 않게 해줘” 좋음
  • “왼쪽 30%는 제목을 넣을 수 있게 비워줘” 좋음

수정용 템플릿:

이전 이미지의 [유지할 요소]는 그대로 두고, [바꿀 요소]를 [원하는 방향]으로 수정해줘. [제외할 요소]는 넣지 마.

예시:

이전 이미지의 따뜻한 카페 분위기는 유지하고, 창밖 배경을 비 오는 밤거리로 바꿔줘. 컵은 더 크게 보이게 중앙 하단에 배치하고, 상단에는 제목을 넣을 수 있는 빈 공간을 남겨줘. 글자와 로고는 넣지 마.

무료로 AI 그림 프롬프트 연습하는 법

처음부터 유료 도구를 결제할 필요는 없습니다. 무료로 이용할 수 있거나 접근 조건을 확인할 수 있는 도구에서 프롬프트 구조를 익힌 뒤, 필요한 경우 고급 도구로 넘어가는 방식이 좋습니다.

가장 쉬운 연습 순서는 다음과 같습니다.

  1. Microsoft Copilot 또는 Microsoft Designer의 이미지 생성 기능에 접속해 필요한 경우 Microsoft 계정으로 로그인합니다.
  2. 입력창에 “정사각형 블로그 썸네일용”처럼 용도를 먼저 씁니다.
  3. 주제, 배경, 스타일, 구도, 조명, 제외 요소를 한 문장으로 입력합니다.
  4. 결과 이미지를 비교해 마음에 드는 이미지를 고릅니다.
  5. “배경을 단순하게”, “제품을 더 크게”, “텍스트 공간을 확보”처럼 수정 요청을 합니다.
  6. 최종 이미지를 다운로드하고, 필요한 경우 Canva 같은 디자인 도구에서 실제 텍스트를 얹습니다.

무료 또는 기본 제공 도구는 생성 속도, 해상도, 하루 사용량, 상업적 사용 조건이 바뀔 수 있습니다. Microsoft Copilot도 일부 기능은 로그인해야 접근할 수 있고 사용량 제한이 있을 수 있습니다. 특히 업무나 판매용으로 쓸 이미지는 각 서비스의 이용 약관을 확인하는 것이 안전합니다.

결과가 너무 평범할 때

프롬프트에 구도, 조명, 색감, 질감이 빠졌을 가능성이 큽니다.

수정 전:

강아지 그림

수정 후:

작은 갈색 강아지가 창가 쿠션 위에서 잠들어 있는 장면, 따뜻한 오후 햇살, 부드러운 털 질감, 아늑한 거실 배경, 수채화 일러스트 스타일, 파스텔톤, 정사각형 구도

글자가 깨질 때

AI 이미지 생성 도구는 이미지 안의 정확한 한글 텍스트에 약한 경우가 많습니다. 포스터나 썸네일을 만들 때는 이미지 안에 글자를 직접 생성하게 하지 말고, 빈 공간만 확보한 뒤 편집 도구에서 텍스트를 넣는 방식이 안정적입니다.

프롬프트 예시:

상단에 제목을 넣을 수 있는 넓은 빈 공간, 이미지 안에는 글자 없음, 로고 없음

손이나 얼굴이 어색할 때

복잡한 손동작, 여러 명의 인물, 작은 얼굴이 많을수록 오류가 늘어납니다. 초보자는 다음처럼 단순화하세요.

  • 손을 화면 밖으로 설정
  • 인물은 1명만 배치
  • 얼굴 클로즈업 또는 뒷모습 활용
  • 복잡한 포즈 대신 자연스러운 정면 자세 사용

예시:

인물의 손은 화면 밖에 있고, 자연스럽게 미소 짓는 정면 상반신 구도, 단순한 배경

AI 그림 프롬프트 작성 시 고르는 기준과 주의점

좋은 프롬프트는 길다고 좋은 것이 아닙니다. 필요한 정보가 빠짐없이 들어간 짧고 명확한 문장이 좋습니다.

체크리스트는 다음과 같습니다.

  • 이미지 용도를 썼는가?
  • 비율 또는 구도를 지정했는가?
  • 주 피사체가 명확한가?
  • 배경이 구체적인가?
  • 스타일을 1~2개로 제한했는가?
  • 조명과 색감을 넣었는가?
  • 제외할 요소를 적었는가?
  • 텍스트가 필요하면 직접 생성하지 않고 여백만 요청했는가?
  • 상업적 사용 전 이용 약관을 확인했는가?

또한 유명 캐릭터, 브랜드 로고, 실제 인물의 초상, 특정 작가의 화풍을 그대로 복제하는 요청은 저작권·초상권 문제가 생길 수 있습니다. 안전하게 쓰려면 “특정 캐릭터처럼”보다 “귀여운 동물 마스코트”, “복고풍 만화 느낌”처럼 일반화해 작성하세요.

바로 복사해서 쓰는 프롬프트 예시

블로그 썸네일용

AI 기술을 설명하는 블로그 썸네일 이미지, 노트북과 빛나는 회로 패턴이 있는 책상, 파란색과 보라색 그라데이션 배경, 미니멀한 3D 일러스트 스타일, 중앙에 주요 오브젝트, 상단에는 제목을 넣을 수 있는 빈 공간, 선명하고 깔끔한 이미지, 글자와 로고 없음

제품 광고용

프리미엄 텀블러 제품 광고 이미지, 무광 스테인리스 텀블러 1개가 밝은 회색 배경 앞에 놓여 있음, 스튜디오 조명, 부드러운 그림자, 중앙 배치, 고급스럽고 미니멀한 상업 사진 스타일, 제품 라벨은 비워두기, 로고와 글자 없음

캐릭터 콘셉트용

작은 우주 탐험가 고양이 캐릭터, 둥근 헬멧과 작은 배낭을 착용, 달 표면 위에 서 있는 장면, 귀여운 3D 캐릭터 스타일, 큰 눈, 파스텔 색감, 정면 전신 구도, 단순한 배경, 선명한 렌더링, 글자 없음

FAQ

Q1. AI 그림 프롬프트는 한국어로 써도 되나요?

네, 대부분의 최신 이미지 생성 도구는 한국어 입력을 어느 정도 이해합니다. 다만 특정 스타일, 카메라 용어, 구도 표현은 영어가 더 안정적으로 반영되는 경우도 있습니다. 초보자는 한국어로 시작하고, 필요한 경우 “close-up, studio lighting, minimal background” 같은 핵심 영어 표현만 섞어도 충분합니다.

Q2. 프롬프트는 길수록 좋은가요?

무조건 길다고 좋은 것은 아닙니다. 주제, 배경, 스타일, 구도, 조명, 색감, 제외 요소가 들어가면 충분합니다. 너무 많은 스타일과 조건을 넣으면 이미지가 산만해질 수 있습니다. 처음에는 2~4문장 정도로 작성하고, 결과를 보며 수정하는 방식이 좋습니다.

Q3. 상업용 이미지로 사용해도 되나요?

도구마다 다릅니다. 무료 플랜과 유료 플랜의 상업적 사용 조건, 저작권 정책, 생성물 사용 범위가 다를 수 있으므로 반드시 해당 서비스의 이용 약관을 확인해야 합니다. 특히 브랜드 로고, 유명 캐릭터, 실제 인물과 닮은 이미지는 별도 권리 문제가 생길 수 있어 주의가 필요합니다.

결론: 좋은 AI 그림은 좋은 입력 순서에서 나온다

AI 그림 프롬프트 작성법의 핵심은 “멋지게 그려줘”가 아니라 무엇을, 어디에, 어떤 스타일과 구도로, 어떤 빛과 색감으로 만들지를 구체적으로 알려주는 것입니다.

처음에는 아래 공식만 기억해도 충분합니다.

주제 + 상황 + 배경 + 스타일 + 구도 + 조명 + 색감 + 제외 요소

그리고 한 번에 완성하려 하지 말고, 생성 결과를 보며 “유지할 것”과 “바꿀 것”을 나눠 수정하세요. 이 방식만 익혀도 블로그 썸네일, 광고 시안, 캐릭터 콘셉트, SNS 이미지까지 훨씬 안정적으로 만들 수 있습니다.
