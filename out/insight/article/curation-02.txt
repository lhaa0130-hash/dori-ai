## 왜 프롬프트 템플릿인가

AI를 쓰는 사람과 잘 쓰는 사람의 차이는 대부분 **프롬프트 품질**에서 납니다. "요약해줘"와 "다음 글을 핵심 3가지로 요약하되, 각 항목을 2줄 이내로 쓰고, 전문 지식이 없는 30대 직장인이 이해할 수 있는 언어로 작성해줘"는 결과물이 완전히 다릅니다.

이 큐레이션은 **지금 바로 복붙해서 쓸 수 있는 프롬프트 템플릿 30가지**를 엄선했습니다. `[대괄호]` 안의 내용만 바꿔서 쓰세요.

## 📝 글쓰기 & 문서 작성 (1~8)

**① 보고서 초안 작성**
```
[주제]에 대한 업무 보고서를 작성해줘.
- 대상 독자: [직급/부서]
- 분량: A4 1장 분량
- 포함 항목: 현황, 문제점, 원인 분석, 해결 방안, 기대 효과
- 톤: 간결하고 전문적으로
```

**② 이메일 초안 (정중한 거절)**
```
다음 상황에 맞는 정중한 거절 이메일을 써줘.
상황: [상황 설명]
보내는 사람: [나의 역할]
받는 사람: [상대방 역할]
거절 이유: [이유]
대안 제시 여부: [있음/없음]
```

**③ SNS 게시물 작성**
```
[플랫폼: 인스타그램/링크드인/트위터]에 올릴 게시물을 써줘.
주제: [내용]
톤: [캐주얼/전문적/유머러스]
해시태그: [5개 추천]
길이: [플랫폼 최적 길이]
```

**④ 회의록 정리**
```
아래 회의 내용을 공식 회의록 형식으로 정리해줘.
- 날짜·참석자 포함
- 결정사항과 미결사항 구분
- 담당자와 마감일이 있으면 명시
- 다음 회의 안건 제안

[회의 내용 붙여넣기]
```

**⑤ 제안서 구조 잡기**
```
[서비스/제품명]을 [대상 고객]에게 제안하는 제안서 목차를 잡아줘.
제안 목적: [목적]
차별점: [주요 강점 3가지]
예산 범위: [금액]
제안서 분량: [페이지 수]
```

**⑥ 블로그 글 작성**
```
다음 키워드로 검색 최적화된 블로그 글을 써줘.
키워드: [키워드]
타겟 독자: [독자 설명]
글의 목적: [정보 제공/설득/경험 공유]
분량: [약 1,500자]
구조: 도입-본문(소제목 3개)-결론 형식
```

**⑦ 보도자료 작성**
```
[회사명]의 [발표 내용]에 대한 보도자료를 작성해줘.
발표일: [날짜]
주요 내용: [핵심 사항]
인용구: [대표자 이름과 직함]의 코멘트 포함
배포 대상: [언론사 종류]
```

**⑧ 약관·정책 문서 요약**
```
다음 약관/개인정보처리방침을 일반인이 이해하기 쉽게 요약해줘.
- 가장 중요한 5가지 항목 먼저
- 사용자에게 불리할 수 있는 조항 별도 강조
- 전문용어는 괄호 안에 쉬운 말로 설명

[문서 내용 붙여넣기]
```
