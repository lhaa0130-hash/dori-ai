노트북LM(NotebookLM) 사용법: 자료 업로드부터 요약·질문·오디오까지 단계별 가이드

2026년 6월 26일AM 08:01 by illo
공유하기

노트북LM(NotebookLM) 사용법을 검색했다면 아마 “PDF나 강의자료를 넣으면 AI가 요약해 준다는데, 어디서부터 눌러야 하지?”가 가장 궁금할 것입니다. NotebookLM은 구글이 제공하는 AI 기반 노트·리서치 도구로, 사용자가 넣은 자료를 바탕으로 요약, 질의응답, 핵심 정리, 학습 가이드, 오디오 개요 등을 만들어 줍니다.

핵심은 간단합니다. 노트북을 만들고 자료를 추가하고 질문하거나 요약을 생성하고 필요한 답을 노트로 저장하면 됩니다. 아래 순서대로 따라 하면 처음 사용하는 사람도 바로 업무, 공부, 리서치에 활용할 수 있습니다.

1단계: 노트북LM(NotebookLM) 접속 및 로그인하기

먼저 브라우저에서 NotebookLM에 접속합니다. 일반적으로 구글 계정으로 로그인해 사용할 수 있으며, 별도 프로그램 설치 없이 웹에서 작동합니다.

  • 접속: notebooklm.google.com
  • 로그인: 개인 Google 계정 또는 조직에서 허용한 Google Workspace 계정
  • 권장 브라우저: Chrome, Edge 등 최신 브라우저

로그인 후 화면에 새 노트북 만들기, Create new, New notebook과 비슷한 버튼이 보입니다. UI 문구는 업데이트에 따라 조금씩 달라질 수 있지만, 기본 흐름은 같습니다.

처음 쓰는 경우에는 업무용 기밀 문서보다 공개 자료나 개인 학습 자료로 먼저 테스트하는 것을 추천합니다.

2단계: 새 노트북 만들기

NotebookLM에서 하나의 노트북은 특정 주제별 작업 공간이라고 보면 됩니다. 예를 들어 “마케팅 리포트 분석”, “논문 읽기”, “자격증 공부”, “회의록 정리”처럼 주제별로 나누면 관리가 쉽습니다.

사용 순서는 다음과 같습니다.

  • 새 노트북 만들기 버튼 클릭
  • 노트북 제목 입력
    예: “2026년 상반기 마케팅 자료 분석”
  • 생성된 노트북 화면으로 이동
  • 왼쪽 또는 중앙의 소스 추가 영역 확인

처음부터 제목을 완벽하게 지을 필요는 없습니다. 나중에 변경할 수 있으므로, 자료의 주제만 알아볼 수 있게 적으면 충분합니다.

3단계: 자료 소스 추가하기

노트북LM(NotebookLM) 사용법에서 가장 중요한 단계가 바로 소스 추가입니다. NotebookLM은 일반 챗봇처럼 인터넷 전체를 마음대로 뒤지는 방식이 아니라, 사용자가 넣은 자료를 중심으로 답변합니다. 그래서 좋은 자료를 넣을수록 답변 품질이 좋아집니다.

대표적으로 추가할 수 있는 자료 유형은 다음과 같습니다.

소스 유형 활용 예시
PDF 논문, 보고서, 전자책, 강의자료
Google Docs 회의록, 원고, 업무 문서
Google Slides 강의안, 발표자료
웹사이트 URL 기사, 공식 문서, 블로그 글
복사한 텍스트 메모, 이메일 초안, 인터뷰 원문
YouTube URL 강의·인터뷰 영상 내용 분석에 활용 가능

자료 추가 방법은 보통 다음과 같습니다.

  • 노트북 안에서 소스 추가 클릭
  • 업로드할 유형 선택
  • 파일 업로드 또는 URL 입력
  • 자료 처리가 끝날 때까지 대기
  • 왼쪽 소스 목록에 자료가 표시되는지 확인

여러 자료를 넣을 때는 같은 주제끼리 묶는 것이 좋습니다. 예를 들어 “논문 5개 + 관련 기사 3개 + 내 메모 1개”를 한 노트북에 넣으면, NotebookLM이 자료 간 공통점과 차이점을 비교하는 데 유리합니다.

4단계: 자동 요약과 소스 가이드 확인하기

소스를 추가하면 NotebookLM이 자료의 핵심을 요약해 줍니다. 화면에는 자료별 요약, 주요 주제, 추천 질문, 학습 가이드와 비슷한 항목이 표시될 수 있습니다.

이때 바로 확인할 것은 세 가지입니다.

  • 자료 요약이 제대로 되었는지
  • 문서의 핵심 주제가 맞게 잡혔는지
  • 추천 질문이 내가 궁금한 방향과 맞는지

예를 들어 80쪽짜리 PDF 보고서를 넣었다면, 먼저 이렇게 물어볼 수 있습니다.

  • “이 보고서의 핵심 주장을 5줄로 요약해 줘.”
  • “실무자가 봐야 할 부분만 정리해 줘.”
  • “수치, 사례, 결론을 표로 정리해 줘.”
  • “이 자료에서 저자가 반복해서 강조하는 내용을 알려 줘.”

NotebookLM의 장점은 답변에 출처 근거를 함께 보여주는 경우가 많다는 점입니다. 답변 옆의 인용 표시나 출처를 눌러 실제 자료의 어느 부분에서 나온 내용인지 확인하세요.

5단계: 질문 입력으로 원하는 답 뽑아내기

NotebookLM은 단순 요약보다 질문을 잘 입력했을 때 가치가 커집니다. 질문창에 자연어로 입력하면 되지만, 막연하게 묻는 것보다 목적을 함께 적는 것이 좋습니다.

좋은 질문 예시는 다음과 같습니다.

  • “이 자료를 처음 읽는 대학생에게 설명하듯 쉽게 요약해 줘.”
  • “회의에서 바로 말할 수 있게 핵심 쟁점 3개와 근거를 정리해 줘.”
  • “찬성 의견과 반대 의견을 나눠서 표로 비교해 줘.”
  • “시험 대비용으로 예상문제 10개와 정답을 만들어 줘.”
  • “이 문서에서 실행해야 할 액션 아이템만 뽑아 줘.”
  • “자료 간 내용이 충돌하는 부분이 있으면 알려 줘.”

업무용으로 쓸 때는 결과물 형식을 명확히 지정하세요.

  • “표로 정리해 줘”
  • “보고서 문체로 작성해 줘”
  • “메일 초안으로 바꿔 줘”
  • “임원 보고용 1페이지 요약으로 만들어 줘”
  • “초등학생도 이해할 정도로 쉽게 설명해 줘”

이렇게 요청하면 답변을 다시 편집하는 시간이 줄어듭니다.

6단계: 마음에 드는 답변을 노트로 저장하기

NotebookLM에서 생성된 답변은 그대로 흘려보내지 말고, 중요한 내용은 노트로 저장해 두는 것이 좋습니다. 답변 옆에 있는 저장, 노트에 추가, Pin, Save to note와 비슷한 버튼을 누르면 해당 내용을 노트북 안에 보관할 수 있습니다.

추천 저장 방식은 다음과 같습니다.

  • 핵심 요약 노트
  • 중요한 인용구 노트
  • 회의용 질문 리스트
  • 시험 대비 예상문제
  • 보고서 초안
  • 추가 조사할 항목

노트를 쌓아 두면 NotebookLM을 단순 요약 도구가 아니라 주제별 지식 베이스처럼 사용할 수 있습니다.

7단계: 오디오 개요로 듣는 요약 만들기

NotebookLM의 인기 기능 중 하나는 Audio Overview, 즉 오디오 개요입니다. 자료 내용을 바탕으로 대화형 오디오 요약을 생성해 주는 기능입니다. 긴 문서나 복잡한 주제를 이동 중에 듣고 싶을 때 유용합니다.

일반적인 사용 순서는 다음과 같습니다.

  • 노트북에 자료 추가
  • 오른쪽 또는 상단의 Studio 영역 확인
  • Audio Overview 또는 오디오 개요 생성 클릭
  • 생성 완료까지 대기
  • 재생 버튼을 눌러 듣기

오디오 개요는 특히 다음 상황에 좋습니다.

  • 논문을 읽기 전에 전체 맥락 파악
  • 출퇴근 중 보고서 내용 복습
  • 강의자료를 시험 전 빠르게 정리
  • 새로운 업계 리포트를 가볍게 훑기

다만 오디오 개요는 어디까지나 AI가 만든 설명입니다. 중요한 수치, 법률, 계약 조건, 의학 정보 등은 반드시 원문 출처를 다시 확인해야 합니다.

8단계: 무료로 노트북LM 쓰는 법

NotebookLM은 무료로 기본 기능을 사용할 수 있는 서비스입니다. 구글 계정으로 접속해 노트북을 만들고, 자료를 넣고, 요약과 질문 기능을 활용할 수 있습니다.

무료로 효율적으로 쓰려면 다음처럼 사용하세요.

  • 노트북을 주제별로 나누기
  • 한 노트북에 너무 잡다한 자료를 섞지 않기
  • 질문을 구체적으로 입력하기
  • 생성 결과를 무조건 믿지 말고 출처 확인하기
  • 꼭 필요한 자료만 업로드하기

유료 플랜 또는 조직용 플랜은 제공 국가, 계정 유형, 구글 정책에 따라 달라질 수 있습니다. 일반적으로 유료 버전은 사용 한도 확대, 더 많은 노트북·소스 처리, 고급 공유 및 팀 기능 등이 포함될 수 있으므로, 실제 가격과 포함 기능은 결제 전 공식 NotebookLM 또는 Google One/Workspace 안내 페이지에서 확인하는 것이 안전합니다.

9단계: 자주 막히는 포인트와 해결법

PDF가 제대로 읽히지 않을 때

스캔본 PDF나 이미지 위주의 문서는 AI가 내용을 정확히 읽지 못할 수 있습니다. 이 경우 OCR 처리된 PDF로 변환하거나, 중요한 부분을 텍스트로 복사해 소스로 추가하는 것이 좋습니다.

답변이 너무 일반적일 때

질문이 넓으면 답도 넓어집니다. “요약해 줘”보다 “마케팅 팀장이 다음 회의에서 의사결정할 수 있게 핵심 문제, 근거, 제안으로 나눠 요약해 줘”처럼 역할과 목적을 넣으세요.

한국어 답변이 어색할 때

NotebookLM은 한국어로도 사용할 수 있지만, 원문 언어와 자료 품질에 따라 표현이 달라질 수 있습니다. 답변 끝에 “자연스러운 한국어 보고서 문체로 다시 써 줘”라고 요청하면 개선됩니다.

출처와 다른 내용이 나올 때

AI 답변은 항상 검증이 필요합니다. NotebookLM이 출처 기반으로 답하더라도 요약 과정에서 뉘앙스가 달라질 수 있습니다. 중요한 내용은 답변의 인용 출처를 눌러 원문과 대조하세요.

노트북LM 활용 팁: 공부·업무별 추천 방식

공부용으로는 교재 PDF, 강의안, 필기 내용을 한 노트북에 넣고 “시험에 나올 만한 개념”, “오답노트”, “빈칸 문제”를 만들어 보세요. 특히 어려운 개념은 “중학생도 이해할 수 있게 비유로 설명해 줘”라고 요청하면 좋습니다.

업무용으로는 회의록, 보고서, 고객 인터뷰, 시장조사 자료를 넣고 “의사결정에 필요한 쟁점”, “리스크”, “다음 액션”을 뽑아내면 효과적입니다. 단, 회사 보안 정책상 외부 AI 도구에 업로드하면 안 되는 자료는 절대 넣지 마세요.

콘텐츠 제작자는 자료 여러 개를 넣고 “공통으로 반복되는 주장”, “서로 다른 관점”, “독자가 궁금해할 질문”을 뽑아 글 구조를 만들 수 있습니다. 이때 최종 문장은 반드시 직접 검토해 사실 오류와 과장 표현을 줄여야 합니다.

고르는 기준과 주의점

NotebookLM이 잘 맞는 사람은 자료 기반으로 정확하게 정리하고 싶은 사람입니다. 논문, 보고서, 강의자료, 회의록처럼 원문이 있는 작업에 특히 강합니다.

반대로 최신 뉴스 검색, 실시간 가격 비교, 웹 전체 탐색이 목적이라면 일반 검색엔진이나 실시간 웹 검색형 AI가 더 적합할 수 있습니다. NotebookLM은 “내가 넣은 자료를 깊게 이해하고 정리하는 도구”에 가깝습니다.

사용 전 꼭 기억할 점은 다음과 같습니다.

  • 민감한 개인정보, 계약서, 내부 기밀은 업로드 전 정책 확인
  • AI 답변은 원문 출처로 검증
  • 자료 품질이 낮으면 답변 품질도 낮아짐
  • 한 노트북에는 하나의 주제만 담는 것이 좋음
  • 최종 보고서나 과제는 사람이 다시 편집해야 함

자주 묻는 질문

Q1. 노트북LM(NotebookLM)은 무료인가요?

기본적으로 무료 사용이 가능합니다. 구글 계정으로 접속해 노트북 생성, 자료 추가, 요약, 질문 기능을 사용할 수 있습니다. 다만 사용량 한도, 고급 기능, 조직용 기능은 계정 유형이나 유료 플랜에 따라 달라질 수 있으므로 공식 안내를 확인하는 것이 좋습니다.

Q2. ChatGPT와 NotebookLM은 무엇이 다른가요?

ChatGPT가 범용 대화형 AI에 가깝다면, NotebookLM은 사용자가 넣은 자료를 중심으로 답하는 리서치·노트 도구에 가깝습니다. 특정 PDF, 문서, URL을 근거로 요약하고 질문하려면 NotebookLM이 편리합니다. 반면 폭넓은 아이디어 생성이나 일반 대화는 범용 챗봇이 더 적합할 수 있습니다.

Q3. 노트북LM에 회사 문서를 올려도 되나요?

가능 여부는 회사의 보안 정책에 따라 다릅니다. 외부 AI 서비스에 업로드가 금지된 문서라면 사용하면 안 됩니다. 고객 개인정보, 재무자료, 계약서, 미공개 전략 문서 등은 특히 주의해야 합니다. 조직용 계정이라면 관리자가 설정한 데이터 보호 정책을 먼저 확인하세요.

결론: 노트북LM 사용법의 핵심은 ‘좋은 자료 + 구체적 질문’

노트북LM(NotebookLM) 사용법은 어렵지 않습니다. 노트북 생성 자료 추가 요약 확인 구체적 질문 답변 저장 오디오 개요 활용 순서만 익히면 됩니다. 중요한 것은 AI에게 모든 것을 맡기는 것이 아니라, 신뢰할 수 있는 자료를 넣고 원하는 결과 형식을 분명히 요청하는 것입니다.

공부 자료 정리, 논문 분석, 회의록 요약, 보고서 초안 작성처럼 “근거가 있는 자료를 빠르게 이해해야 하는 작업”이 많다면 NotebookLM은 무료로 시작해 볼 만한 도구입니다. 처음에는 작은 PDF 한두 개로 테스트한 뒤, 자신만의 질문 템플릿을 만들어 꾸준히 활용해 보세요.
