생성형 AI 종류 총정리를 찾는 이유는 보통 하나입니다. **ChatGPT 같은 챗봇만 AI인지**, 이미지·영상·음악·코딩 AI는 어떻게 다른지, 내 업무에는 어떤 AI를 써야 하는지 한 번에 알고 싶기 때문입니다. 이 글에서는 생성형 AI를 기능별로 나누고, 대표 도구와 장단점, 실제 활용 기준까지 정리합니다.

## 생성형 AI란? 종류 총정리 전에 알아야 할 기본 개념

**생성형 AI**는 사용자의 입력을 바탕으로 새로운 콘텐츠를 만들어내는 인공지능입니다. 텍스트를 쓰고, 이미지를 그리고, 영상을 만들고, 음성을 합성하고, 코드를 작성하며, 최근에는 여러 앱을 연결해 실제 업무를 수행하는 **AI 에이전트** 형태로 확장되고 있습니다.

기존 AI가 “분류·예측·추천”에 강했다면, 생성형 AI는 “작성·제작·변환·자동화”에 강합니다. 예를 들어 고객 리뷰를 긍정/부정으로 나누는 것은 전통적 AI에 가깝고, 그 리뷰를 바탕으로 답변 메일을 작성하는 것은 생성형 AI에 가깝습니다.

## 생성형 AI 종류 총정리: 한눈에 보는 분류표

| 종류 | 만드는 결과물 | 대표 도구 예시 | 추천 대상 |
|---|---|---|---|
| 텍스트 생성 AI | 글, 요약, 번역, 기획안 | **ChatGPT**, **Claude**, **Gemini** | 직장인, 학생, 마케터 |
| 이미지 생성 AI | 일러스트, 광고 이미지, 콘셉트 아트 | **Midjourney**, **DALL·E**, **Adobe Firefly**, **Stable Diffusion** | 디자이너, 쇼핑몰, 콘텐츠 제작자 |
| 영상 생성 AI | 짧은 영상, 광고 컷, B-roll | **Runway**, **Pika**, **Kling** | 영상 제작자, 브랜드 마케터 |
| 음성·음악 생성 AI | 내레이션, 더빙, 음악 | **ElevenLabs**, **Suno**, **Udio** | 크리에이터, 교육자, 팟캐스터 |
| 코딩 생성 AI | 코드, 테스트, 디버깅 | **GitHub Copilot**, **Cursor**, **Replit AI** | 개발자, 스타트업, 기획자 |
| 멀티모달 AI | 텍스트+이미지+음성+파일 이해 | **GPT 계열**, **Gemini**, **Claude** | 복합 업무 사용자 |
| AI 에이전트 | 검색, 실행, 앱 연동, 자동화 | **Zapier AI**, **Make**, **MCP 기반 연동** | 운영팀, 세일즈, 자동화 담당자 |

## 생성형 AI 종류별 자세한 설명

### 1. 텍스트 생성 AI: 가장 범용적인 생성형 AI 종류

텍스트 생성 AI는 질문에 답하고, 글을 쓰고, 문서를 요약하고, 번역하고, 회의록을 정리하는 데 쓰입니다. 대표적으로 **ChatGPT**, **Claude**, **Gemini**가 있습니다.

**ChatGPT**는 범용성이 높아 아이디어 정리, 글쓰기, 표 만들기, 간단한 데이터 분석에 많이 사용됩니다. OpenAI는 2026년 1월 16일 월 **$8**의 저가형 요금제인 **ChatGPT Go**를 전 세계에 출시했습니다. 다만 국가별 제공 여부와 요금제 구성은 바뀔 수 있으므로 가입 전 공식 화면 확인이 필요합니다.

**Claude**는 긴 문서 읽기, 논리적 글쓰기, 업무 문서 정리에 강점이 있는 편입니다. **Claude Pro 요금제는 월 $20**로 제공되며, 세부 기능과 사용량 제한은 플랜과 지역, 시점에 따라 달라질 수 있습니다. 외부 도구나 커넥터 연동 기능은 서비스 정책과 지원 환경을 공식 문서에서 확인하는 것이 좋습니다.

**Gemini**는 Google 생태계와의 연결성이 강점입니다. 문서, 메일, 검색, 이미지 이해 등 Google 서비스와 함께 쓰려는 사용자에게 적합합니다.

- 장점: 활용 범위가 가장 넓고 진입장벽이 낮음  
- 단점: 사실 오류, 출처 누락, 그럴듯한 오답 가능  
- 추천 대상: 보고서·메일·기획안·요약 업무가 많은 사람  

### 2. 이미지 생성 AI: 그림·광고·콘셉트 제작용 AI

이미지 생성 AI는 프롬프트를 입력하면 새로운 이미지를 만들어줍니다. 대표 도구는 **Midjourney**, **DALL·E**, **Adobe Firefly**, **Stable Diffusion**입니다.

**Midjourney**는 감각적인 비주얼과 콘셉트 아트에 강합니다. 브랜드 무드보드, 포스터 시안, 게임·웹소설 이미지 아이디어를 만들 때 자주 쓰입니다.

**DALL·E**는 대화형 이미지 생성과 편집에 적합합니다. 텍스트로 “배경만 바꿔줘”, “제품을 더 크게 보여줘”처럼 요청하는 방식이 직관적입니다.

**Adobe Firefly**는 Adobe 생태계와 연결되는 점이 장점입니다. Photoshop, Illustrator 등 디자인 워크플로에 AI 이미지를 넣고 싶은 사용자에게 유용합니다.

**Stable Diffusion**은 오픈소스 기반 모델로 널리 알려져 있으며, 로컬 환경이나 커스텀 모델을 활용하려는 고급 사용자에게 적합합니다.

- 장점: 시안 제작 속도가 빠르고 비전문가도 이미지 제작 가능  
- 단점: 손·문자·로고 표현 오류, 저작권·상표권 검토 필요  
- 추천 대상: 디자이너, 쇼핑몰 운영자, SNS 콘텐츠 제작자  

### 3. 영상 생성 AI: 짧은 광고와 숏폼 제작에 강한 종류

영상 생성 AI는 텍스트나 이미지를 입력해 짧은 영상을 생성하거나, 기존 영상을 편집·확장하는 도구입니다. 대표적으로 **Runway**, **Pika**, **Kling** 등이 거론됩니다.

**Runway**는 영상 생성, 배경 제거, 스타일 변환, 영상 편집 기능으로 알려져 있습니다. 광고 시안, 뮤직비디오 콘셉트, B-roll 제작에 활용됩니다.

**Pika**는 짧은 클립 생성과 스타일화된 영상 제작에 자주 사용됩니다. SNS용 짧은 영상 실험에 적합합니다.

**Kling**은 사실적인 움직임과 장면 생성 측면에서 관심을 받고 있는 영상 생성 AI 중 하나입니다. 다만 영상 AI는 모델과 요금제, 사용 가능 지역이 자주 바뀌므로 실제 제작 전 현재 조건을 확인해야 합니다.

- 장점: 촬영 전 콘티·시안 제작에 강함  
- 단점: 긴 영상의 일관성, 인물 디테일, 상업적 권리 확인 필요  
- 추천 대상: 마케터, 영상 편집자, 광고 기획자  

### 4. 음성·음악 생성 AI: 내레이션, 더빙, 배경음악 제작

음성 생성 AI는 텍스트를 자연스러운 음성으로 바꾸고, 음악 생성 AI는 프롬프트로 노래나 배경음악을 만듭니다. 대표 도구로 **ElevenLabs**, **Suno**, **Udio**가 있습니다.

**ElevenLabs**는 자연스러운 음성 합성, 더빙, 보이스오버 제작에 많이 언급됩니다. 교육 영상, 유튜브 내레이션, 팟캐스트 제작에 활용할 수 있습니다.

**Suno**와 **Udio**는 가사와 장르를 입력해 음악을 생성하는 도구로 알려져 있습니다. 브랜드송 시안, 숏폼 배경음악, 데모 음악 제작에 사용할 수 있습니다.

- 장점: 녹음실 없이 빠르게 음성·음악 시안 제작 가능  
- 단점: 목소리 복제 동의, 음악 저작권, 상업적 이용 조건 확인 필요  
- 추천 대상: 유튜버, 강의 제작자, 팟캐스터, 게임 제작자  

### 5. 코딩 생성 AI: 개발 생산성을 높이는 생성형 AI 종류

코딩 AI는 코드를 자동완성하고, 오류를 설명하고, 테스트 코드를 만들고, 리팩터링을 도와줍니다. 대표적으로 **GitHub Copilot**, **Cursor**, **Replit AI**가 있습니다.

**GitHub Copilot**은 IDE 안에서 코드 자동완성과 함수 제안을 제공하는 개발 보조 도구입니다. 반복 코드 작성, 테스트 코드 생성, API 사용 예시 작성에 유용합니다.

**Cursor**는 AI 기능을 중심에 둔 코드 에디터로 알려져 있습니다. 코드베이스를 읽고 수정 제안을 하거나, 여러 파일을 함께 다루는 작업에 적합합니다.

**Replit AI**는 브라우저 기반 개발 환경에서 AI 도움을 받을 수 있어 초보자나 빠른 프로토타입 제작에 유리합니다.

- 장점: 반복 개발 속도 향상, 초보자의 학습 보조  
- 단점: 보안 취약 코드, 라이선스 문제, 잘못된 로직 가능  
- 추천 대상: 개발자, 데이터 분석가, 노코드에서 코드로 넘어가는 사용자  

### 6. 멀티모달 AI: 텍스트·이미지·음성·파일을 함께 이해

멀티모달 AI는 한 가지 입력만 처리하지 않고, 텍스트·이미지·음성·문서 파일 등을 함께 이해합니다. 예를 들어 사진을 올리고 “이 제품 상세페이지의 문제점을 찾아줘”라고 하거나, PDF를 업로드하고 “핵심 쟁점만 표로 정리해줘”라고 요청할 수 있습니다.

최근 주요 AI 서비스는 단순 챗봇에서 멀티모달 업무 도구로 이동하고 있습니다. 사용자는 문서 요약, 이미지 분석, 데이터 해석, 음성 대화까지 하나의 인터페이스에서 처리하기를 원하기 때문입니다.

- 장점: 여러 업무를 한 도구에서 처리 가능  
- 단점: 파일 보안, 업로드 제한, 답변 검증 필요  
- 추천 대상: 리서처, 기획자, 컨설턴트, 사무직  

### 7. AI 에이전트: 생성형 AI의 다음 단계

AI 에이전트는 단순히 답변만 생성하는 것이 아니라, 목표를 이해하고 여러 단계를 계획한 뒤 실제 도구를 호출해 실행하는 AI입니다. 예를 들어 “지난달 매출 데이터를 보고 이상치를 찾아 Slack에 보고해줘” 같은 요청을 처리하는 방향입니다.

대표적으로 **Zapier AI**, **Make** 같은 자동화 플랫폼, 그리고 **MCP**를 통한 앱 연결 방식이 주목받고 있습니다. 다만 MCP 기반 연동이나 커넥터 지원 범위는 도구, 플랜, 배포 환경에 따라 달라질 수 있으므로 실제 업무에 적용하기 전 공식 문서와 보안 조건을 확인해야 합니다.

- 장점: 반복 업무 자동화, 앱 간 연결, 업무 실행 가능  
- 단점: 권한 관리 실패 시 보안 위험, 잘못된 실행 가능  
- 추천 대상: 운영팀, 세일즈팀, CS팀, 자동화 담당자  

## 실제 예시로 보는 생성형 AI 활용처

생성형 AI는 “무엇을 만들 것인가”에 따라 선택해야 합니다.

마케터라면 텍스트 AI로 광고 문구를 만들고, 이미지 AI로 배너 시안을 만들고, 영상 AI로 숏폼 초안을 만들 수 있습니다. 개발자는 코딩 AI로 반복 코드를 줄이고, 문서화와 테스트 코드를 자동화할 수 있습니다. 교육자는 텍스트 AI로 강의안을 만들고, 음성 AI로 내레이션을 제작할 수 있습니다. 쇼핑몰 운영자는 제품 설명, 상세페이지 이미지 시안, 고객 문의 답변 초안을 만들 수 있습니다.

중요한 점은 생성형 AI가 최종 책임자가 아니라는 것입니다. 초안 제작에는 강하지만, 사실 확인·법적 검토·브랜드 톤 조정은 사람이 해야 합니다.

## 생성형 AI를 고를 때 기준

### 1. 결과물 기준으로 고르기

글이 필요하면 **텍스트 AI**, 이미지가 필요하면 **이미지 AI**, 영상이 필요하면 **영상 AI**, 반복 실행이 필요하면 **AI 에이전트**를 고르는 것이 기본입니다. 처음부터 모든 기능이 있는 비싼 도구를 찾기보다, 가장 자주 쓰는 작업 하나를 기준으로 선택하는 편이 효율적입니다.

### 2. 무료 여부보다 사용량과 제한을 확인하기

많은 AI 도구가 무료 플랜이나 무료 체험을 제공하지만, 생성 횟수·속도·상업적 이용 조건·고급 모델 접근 여부가 다릅니다. 특히 팀에서 쓸 경우 개인 무료 계정보다 관리 기능, 보안, 데이터 사용 정책이 더 중요합니다.

### 3. 보안과 저작권을 확인하기

회사 내부 문서, 고객 정보, 계약서, 미공개 코드 등을 아무 AI에나 넣으면 위험합니다. 업무용으로 사용할 때는 입력 데이터가 모델 학습에 사용되는지, 관리자 통제가 가능한지, 로그를 삭제할 수 있는지 확인해야 합니다.

이미지·음악·음성 AI는 상업적 이용 조건도 중요합니다. “생성 가능”과 “상업적으로 안전하게 사용 가능”은 다릅니다.

### 4. 최신 모델보다 워크플로 적합성이 중요

최신 모델이 항상 최고의 선택은 아닙니다. 보고서 작성자는 긴 문서 처리 능력이 중요하고, 디자이너는 이미지 품질과 편집 기능이 중요하며, 개발자는 IDE 통합과 코드베이스 이해 능력이 중요합니다.

최근 AI 모델 가격은 전반적으로 하락하는 추세로 확인됩니다. 고성능 모델의 API 비용이 낮아지고, 특정 업무에 최적화된 경량 모델과 저가형 플랜도 늘어나면서 선택지가 넓어지고 있습니다. 즉, 앞으로는 “가장 비싼 모델”보다 “내 업무에 맞는 모델 조합”이 더 중요해질 가능성이 큽니다.

## 자주 묻는 질문 FAQ

### Q1. 생성형 AI 종류 중 처음 써볼 만한 것은 무엇인가요?

처음이라면 **텍스트 생성 AI**부터 추천합니다. ChatGPT, Claude, Gemini 같은 도구는 글쓰기, 요약, 번역, 아이디어 정리 등 대부분의 업무에 바로 적용할 수 있습니다. 이후 필요에 따라 이미지 AI, 영상 AI, 코딩 AI로 확장하면 됩니다.

### Q2. 생성형 AI와 일반 AI는 어떻게 다른가요?

일반 AI는 데이터를 분류하거나 예측하는 데 주로 쓰입니다. 반면 생성형 AI는 글, 이미지, 영상, 음성, 코드처럼 새로운 결과물을 만듭니다. 예를 들어 “고객 이탈 가능성을 예측”하면 일반 AI에 가깝고, “이탈 고객에게 보낼 맞춤형 이메일을 작성”하면 생성형 AI에 가깝습니다.

### Q3. 무료 생성형 AI만 써도 충분한가요?

개인 학습이나 가벼운 업무에는 무료 플랜만으로도 충분할 수 있습니다. 하지만 사용량이 많거나, 고급 모델이 필요하거나, 회사 데이터를 다루거나, 상업적 콘텐츠를 제작한다면 유료 플랜과 이용 약관을 검토해야 합니다. 특히 업무용은 가격보다 **보안·저작권·관리 기능**이 더 중요합니다.

## 결론: 생성형 AI 종류 총정리의 핵심

생성형 AI는 크게 **텍스트, 이미지, 영상, 음성·음악, 코딩, 멀티모달, AI 에이전트**로 나눌 수 있습니다. 가장 중요한 선택 기준은 “유명한 AI인가”가 아니라 “내가 만들 결과물이 무엇인가”입니다.

글과 문서가 중심이면 텍스트 AI, 시각 콘텐츠가 필요하면 이미지·영상 AI, 개발 업무는 코딩 AI, 반복 실행과 앱 연결이 필요하면 AI 에이전트를 고르면 됩니다. 생성형 AI는 이미 단순한 챗봇을 넘어 업무 도구이자 제작 도구로 확장되고 있습니다. 다만 결과물의 사실성, 저작권, 개인정보, 보안은 반드시 사람이 최종 확인해야 합니다.
