## 들어가며: 반복 업무는 자동화의 적이다

매주 하는 일이 있다. 보고서를 정해진 형식에 맞게 취합하거나, 특정 조건의 이메일이 오면 담당자에게 알리거나, SNS에 정해진 시간에 게시물을 올리거나. 이런 업무들은 한 번 하는 데 30분~1시간이 걸린다. 그리고 매주 반복된다.

1년이면 40~80시간. AI 자동화를 적용하면 이 시간이 사라진다. 처음 설정에 2~4시간을 쓰면, 이후에는 시스템이 알아서 한다.

이 가이드는 AI 자동화의 수준을 4단계로 나누고, 각 단계에서 실제로 구축할 수 있는 파이프라인을 구체적으로 다룬다.

## AI 자동화 4레벨 로드맵

### 레벨 1: 단순 반복 자동화 (노코드)
**목표:** 정해진 조건에 정해진 작업을 자동 실행
**도구:** Zapier, Make (구 Integromat)
**난이도:** 코딩 불필요

**예시:**
- 구글 폼 응답이 오면 → 스프레드시트에 저장 + 담당자에게 이메일 발송
- 특정 키워드 이메일이 오면 → Slack 채널에 알림
- 새 트윗이 올라오면 → 인스타그램 스토리에 자동 공유

### 레벨 2: AI 판단이 포함된 자동화
**목표:** 조건에 따라 AI가 판단하고 다른 액션을 취함
**도구:** Make + OpenAI API, Zapier + ChatGPT
**난이도:** 약간의 설정 능력 필요

**예시:**
- 고객 이메일이 오면 → AI가 문의 유형 분류 → 담당 팀에 자동 배정
- 뉴스 RSS 피드 수집 → AI가 관련성 판단 → 중요도 높은 것만 요약·전달
- SNS 댓글 수집 → AI가 감성 분석 → 부정 댓글만 관리자에게 알림

### 레벨 3: AI 콘텐츠 생성 자동화
**목표:** AI가 콘텐츠를 생성해서 자동 발행
**도구:** n8n + OpenAI/Claude API
**난이도:** 기초 개발 지식 필요

**예시:**
- 매일 오전 7시 트렌드 뉴스 수집 → AI 요약 생성 → 블로그 자동 포스팅
- 상품 정보 입력 → AI 제품 설명 생성 → 쇼핑몰 자동 업로드
- 고객 데이터 → AI 개인화 이메일 초안 생성 → 마케팅 툴로 전송

### 레벨 4: 멀티 에이전트 자동화 시스템
**목표:** 여러 AI가 서로 결과를 주고받으며 복잡한 작업 처리
**도구:** n8n + 여러 AI API + 데이터베이스
**난이도:** 개발/운영 지식 필요

**예시:**
- 시장 리서처 AI → 데이터 수집 → 분석 AI → 보고서 생성 AI → 자동 발송
- 고객 채팅 AI → 대화 저장 → 요약 AI → CRM 자동 업데이트

## 레벨별 시작 가이드

### 레벨 1 시작하기 (Zapier/Make)

**Zapier 30분 시작법:**
1. zapier.com 회원가입 (무료 플랜 100회/월)
2. "Create Zap" 클릭
3. Trigger 선택: "Gmail — New Email" (조건)
4. Filter 추가: 제목에 "[긴급]" 포함 시에만
5. Action 선택: "Slack — Send Message" (실행)
6. 테스트 후 활성화

**첫 번째 Zap 추천:** Google Form → Gmail 자동 응답
- 폼 제출 → AI(ChatGPT 액션)가 응답 이메일 초안 생성 → 자동 발송

### 레벨 2 시작하기 (Make + OpenAI)

**핵심 개념: 모듈 연결**

Make는 레고처럼 모듈을 연결하는 방식이다.

```
[Gmail: 이메일 받기]
    ↓
[OpenAI: 분류 요청]
 - 프롬프트: "이 이메일의 유형을 판단해: 
   A. 구매 문의 B. 기술 지원 C. 환불 요청 D. 기타
   이메일: {{email.body}}"
    ↓
[조건 분기 (Router)]
 - A이면 → 영업팀 Slack 알림
 - B이면 → 기술팀 Slack 알림
 - C이면 → 고객팀 Slack + 환불 처리 폼 링크 첨부
```

### 레벨 3 시작하기 (n8n)

n8n은 자체 서버에 설치하거나 클라우드 버전을 쓸 수 있다. 완전 무료로 시작할 수 있는 자체 설치 버전(Docker)을 추천한다.

**n8n 설치 (Docker):**
```bash
docker run -it --rm \
  --name n8n \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \
  docker.n8n.io/n8nio/n8n
```

설치 후 `localhost:5678`에서 시각적 워크플로우 편집기로 작업한다.

**첫 번째 n8n 워크플로우: 일일 뉴스 브리핑**

```
[Cron: 매일 오전 8시 실행]
    ↓
[HTTP Request: RSS 피드 수집]
 URL: https://feeds.feedburner.com/techcrunch (또는 원하는 피드)
    ↓
[Code: 최신 5개 추출]
    ↓
[OpenAI: 요약 요청]
 "다음 5개 기사를 각각 3줄로 요약하고 
  중요도(상/중/하)를 붙여줘: {{articles}}"
    ↓
[Gmail: 이메일 발송]
 수신: 나의 이메일
 제목: "[AI 브리핑] {{날짜}}"
 본문: AI 요약 내용
```

## 실전 n8n 파이프라인 3가지

### 파이프라인 1: 경쟁사 모니터링 자동화

**비즈니스 가치:** 경쟁사 동향을 매일 자동으로 파악

**흐름:**
```
[Cron: 매일 9시]
    ↓
[HTTP Request] × 3개 (경쟁사 블로그 RSS 각각 수집)
    ↓
[Merge: 3개 결과 합치기]
    ↓
[Code: 날짜 필터 — 어제 이후 글만 선별]
    ↓
[OpenAI]
 - 각 글 요약 (3줄)
 - 전략적 시사점 도출
 - 우리 팀이 대응해야 할 것 있으면 표시
    ↓
[조건 분기]
 - 시사점 있음 → Slack #경쟁사모니터링 채널 발송
 - 없음 → 그냥 로그만 저장
```

**예상 절약 시간:** 주 3시간 → 0시간

### 파이프라인 2: 고객 후기 분석 자동화

**비즈니스 가치:** 고객 불만을 실시간으로 감지

**흐름:**
```
[Cron: 3시간마다]
    ↓
[HTTP Request: 네이버 스마트스토어 / 구글 리뷰 / 크몽 후기 수집]
    ↓
[Code: 마지막 수집 이후 새 후기만 선별]
    ↓
[OpenAI: 감성 분석]
 - 긍정 / 부정 / 중립 분류
 - 부정 리뷰 → 주요 불만 키워드 추출
 - 긍정 리뷰 → 강점 키워드 추출
    ↓
[조건 분기]
 - 부정 리뷰 → Slack #고객불만 + 담당자 멘션
 - 월말 → 전체 분석 보고서 생성 → 이메일 발송
```

### 파이프라인 3: 콘텐츠 자동 발행 시스템

**비즈니스 가치:** 블로그/SNS 콘텐츠를 인력 없이 지속 발행

**흐름:**
```
[Cron: 매일 오전 6시]
    ↓
[HTTP Request: 트렌드 키워드 수집]
 (구글 트렌드 API, 네이버 데이터랩 등)
    ↓
[OpenAI: 콘텐츠 기획]
 "오늘의 트렌드 중 우리 브랜드와 연결할 수 있는 주제를 선정하고
  1500자 분량 블로그 초안을 써줘"
    ↓
[Claude API: 편집]
 "위 초안에서 AI 냄새를 제거하고 자연스럽게 다듬어줘"
    ↓
[WordPress API / GitHub API: 발행]
    ↓
[Slack: 발행 완료 알림]
```

## 자동화 전 반드시 확인해야 할 것

### 1. 자동화할 업무를 먼저 문서화한다

자동화는 현재 하는 작업을 그대로 시스템에 옮기는 것이다. 작업이 모호하면 자동화도 모호하다. 먼저 수작업으로 하는 과정을 단계별로 글로 써본다.

### 2. 에러 처리를 설계한다

자동화 시스템은 반드시 실패한다. 실패했을 때 어떻게 할지 미리 설계한다.
- 실패 시 담당자에게 알림
- 재시도 횟수 설정
- 에러 로그 저장

### 3. 단계별로 테스트한다

전체 파이프라인을 한 번에 실행하지 말고, 각 단계를 개별적으로 테스트한 뒤 연결한다.

### 4. 모니터링을 설정한다

자동화가 잘 돌고 있는지 주기적으로 확인하는 방법을 만든다. 주 1회 실행 로그 확인만으로도 충분하다.

## 비용 계획

| 도구 | 무료 플랜 | 유료 플랜 |
|------|-----------|-----------|
| Zapier | 100회/월 | $19.99/월~ |
| Make | 1,000회/월 | $9/월~ |
| n8n (자체 설치) | 무제한 | 호스팅 서버 비용 |
| OpenAI API | 없음 | 사용량 기반 |
| Claude API | 없음 | 사용량 기반 |

소규모 자동화(월 수백 회 실행)라면 Make 무료 + Claude API로 월 5~10달러 수준으로 시작할 수 있다.

## 마치며

자동화는 한 번에 완성하는 것이 아니다. 작은 것 하나를 자동화하고, 그 시간을 다음 자동화에 쓰는 복리 전략이 효과적이다. 지금 매주 반복하는 업무 목록을 작성하고, 가장 단순한 것 하나부터 시작해보자.
